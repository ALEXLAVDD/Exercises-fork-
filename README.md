
## Δομή του Project

Ο κώδικας είναι χωρισμένος σε 4 υποφακέλους, ένας για κάθε στάδιο της άσκησης:

* **ασκηση 1:** Βασική δημιουργία διεργασιών με χρήση fork().
* **ασκηση 2:** Επικοινωνία γονέα-παιδιού και διαχείριση file descriptors.
* **ασκηση 3:** Χρήση της exec() για φόρτωση νέων προγραμμάτων.
* **ασκηση 4 (Το τελικό πρόγραμμα):** Ένα ολοκληρωμένο σύστημα Κατανομής Εργασιών (Task Dispatcher) που χρησιμοποιεί workers για την παράλληλη καταμέτρηση χαρακτήρων (π.χ. 'a') μέσα σε μεγάλα αρχεία.

## Πώς να το τρέξετε (Άσκηση 4)

Για την τελική άσκηση, παρέχεται ένα Makefile για εύκολη μεταγλώττιση.

1. Μεταγλώττιση:
Ανοίξτε το τερματικό στον φάκελο της Άσκησης 4 και τρέξτε:
make

Αυτό θα δημιουργήσει αυτόματα τα εκτελέσιμα: frontend, dispatcher και worker.

2. Εκτέλεση:
Τρέξτε το frontend δίνοντας ως ορίσματα το αρχείο που θέλετε να διαβάσετε και τον χαρακτήρα που θέλετε να μετρήσετε:
./frontend massive_test.txt a

3. Διαθέσιμες Εντολές Frontend:
Κατά την εκτέλεση, μπορείτε να πληκτρολογήσετε τις εξής εντολές:
* add <N>: Προσθέτει N νέους workers για να βοηθήσουν στο διάβασμα.
* info: Εμφανίζει πληροφορίες για τον dispatcher και τους ενεργούς workers.
* progress: Ζητάει από τον dispatcher να υπολογίσει και να τυπώσει το ποσοστό (%) του αρχείου που έχει διαβαστεί μέχρι στιγμής.
* exit: Τερματίζει ομαλά το πρόγραμμα και όλα τα παιδιά του.

4. Καθαρισμός:
Για να διαγράψετε τα εκτελέσιμα αρχεία, τρέξτε:
make clean
